# Getting Started

## Initial steps

```.sh
poetry init
poetry add openai
poetry add numpy
```

## Add OPENAI_API_KEY

```.sh
export OPENAI_API_KEY=sk-********
```

## Add poetry scripts

```.sh
mkdir src
touch src/main.py
```

&& add config to `pyproject.toml`

## Create virtualenv

```.sh
poetry shell
```
